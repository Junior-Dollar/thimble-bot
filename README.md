# Thimble Bot

A fast and small general-purpose bot, mainly for my own uses.

## Requirements

Node (>=8.6.x).

## Installation

```sh
git clone git@github.com:jozsefsallai/thimble-bot
cd thimble-bot
npm i -g yarn
yarn
yarn setup
```

## Before committing

Make sure lint passes.

```
yarn lint
```
